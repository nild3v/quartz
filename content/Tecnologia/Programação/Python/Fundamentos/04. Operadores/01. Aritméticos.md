## Introdução

Saiba como são os operadores aritméticos básicos em Python

### Soma

```python showCodeRunner
print(1 + 1)
print(2 + 2.5)
print(3 + 3 + 3)
```

### Subtração

```python showCodeRunner
print(1 - 1)
print(2 - 2.5)
print(3 - 3 - 3)
```

### Multiplicação

```python
print(2 * 2)
print(3 * 3 * 3)
print(-2 * 1 * 4)
```

### Divisão

```python showCodeRunner
print(2 / 2)	
print(3 / 2)
print(1.5 / 2)
```

### Divisão Inteira

É uma divisão comum, mas só retorna a parte inteira:

```python showCodeRunner
print(3 // 2)
print(5 // 2)
print(1 // 2)
```

### Módulo (Resto da Divisão)

```python showCodeRunner
print(3 % 2)
print(5 % 2)
print(1 % 2)
```

### Potenciação

```python showCodeRunner
print(2 ** 2)
print(3 ** 2)
print(5 ** 5)
```

### Radiciação

Em Python e em muitas linguagens não há um operador específico para radiciação, mas é possível aplicar essa operação através da potenciação:

$\sqrt[n]{ a } = a^\frac{1}{n}$

Veja:

```python showCodeRunner
print(4 ** (1/2)) # Raiz quadrada de 4
print(9 ** (1/2)) # Raiz quadrada de 9
print(27 ** (1/3)) # Raiz cúbica de 27
```
