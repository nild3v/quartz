## Introdução

Os operadores de comparação são os operadores que permitem realizar uma comparação entre dois valores e retorna um valor booleano que indica se a comparação é válida

### Igual a...

O duplo sinal de igual `==` compara os valores, recomendando quando se trabalha com números, strings, booleanos ou tuplas. Mas também pode ser usado para listas ou dicionários quando se quer comparar os valores:

```python showCodeRunner
print(1 == 1)
print(1 == '1')
print([1, 2] == [1, 2])
print({} == {})
```

A palavra reservada `is` compara os endereços de memória, recomendando quando se trabalha com listas ou dicionários:

```python
print([] is []) # Cada lista possui um enderenço de memória diferente
print({} is {}) # Cada dicionário possui um endereço de memória diferente
```

### Diferente de...

O operador `!=` compara os valores, recomendando quando se trabalha com números, strings e booleanos. Mas também pode ser usado para listas ou dicionários quando se quer comparar os valores:

```python showCodeRunner
print(1 != 2)
print([1, 2] != [1, 2])
print(1 != '1')
print('abc' != 'ABC')
```

A junção das palavras reservadas `not` e `is` compara os endereços de memória, recomendando quando se trabalha com listas ou dicionários:

```python showCodeRunner
print([] is not [])
print({} is not {})
```

### Em...

O operador `in` verifica se um valor é membro de outro, mas o comportamento desse operador é varia para tipos de dados diferentes:

```python showCodeRunner
print(1 in [1, 2, 3]) # Verifica se 1 está incluso na lista
print("a" in ("a", "b", "c")) # Verifica se "a" está incluso na tupla
print(2 in set([1, 3, 4])) # Verifica se 2 está incluso no conjunto
```

```python showCodeRunner
print("a" in {"a": 1, "b": 2}) # Verifica se a chave "a" está definida no dicionário
```

```python showCodeRunner
print("Hello" in "Hello World!") # Verifica se a string inclui a sequência  de caracteres "Hello"
```

### Não Em...

O operador `in` pode ser combinado com o operador de negação `not` para verifica se um valor **não** é membro de outro. Os comportamentos citados anteriormente não mudam:

```python showCodeRunner
print(1 not in [1, 2, 3]) # Verifica se 1 NÃO está incluso na lista
print("a" not in ("a", "b", "c")) # Verifica se "a" NÃO está incluso na tupla
print(2 not in set([1, 3, 4])) # Verifica se 2 NÃO está incluso no conjunto
```

```python showCodeRunner
print("a" not in {"a": 1, "b": 2}) # Verifica se a chave "a" NÃO está definida no dicionário
```

```python showCodeRunner
print("Hello" not in "Hello World!") # Verifica se a string NÃO inclui a sequência  de caracteres "Hello"
```

### Maior que...

```python showCodeRunner

```

### Menor ou igual que...

```python showCodeRunner

print(3 <= 2)
print(4 <= 4)
```

```python showCodeRunner

print(3 >= 2)
```
