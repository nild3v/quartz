## Introdução

Os operadores de atribuição atribui um valor ao operando à sua esquerda baseado no valor do operando à direita

### Atribuição

Atribui um valor à uma variável específica:

```python
x = y
```

```python showCodeRunner
x = 1
y = x

print(x)
print(y)
```

### Atribuição de Adição

Atribui a uma variável $x$ a soma entre ela e uma variável $y$:

```python
x += y # x = x + y
```

```python showCodeRunner
x = 10
x += 5
print(x)
```

### Atribuição de Subtração

Atribui a uma variável $x$ a diferença entre ela e uma variável $y$:

```python
x -= y # x = x - y
```

```python showCodeRunner
x = 1
x -= 1
print(x)
```

### Atribuição de Multiplicação

Atribui a uma variável $x$ o produto entre ela e uma variável $y$:

```python
x *= y # x = x * y
```

```python showCodeRunner
x = 4
x *= 2
print(x)
```

### Atribuição de Divisão

Atribui a uma variável $x$ a divisão entre ela e uma variável $y$:

```python
x /= y # x = x / y
```

```python showCodeRunner
x = 3
x /= 3
print(x)
```

### Atribuição de Divisão Inteira

Atribui a uma variável $x$ a divisão inteira entre ela e uma variável $y$:

```python
x //= y # x = x // y
```

```python showCodeRunner
x = 3
x //= 2
print(x)
```

### Atribuição de Módulo (Resto da Divisão)

Atribui a uma avariável $x$ o módulo entre ela e uma variável $y$:

```python
x %= y # x = x % y
```

```python showCodeRunner
x = 3
x %= 2
print(x)
```

### Atribuição de Exponencial

Atribui a uma variável $x$ o valor dela elevado a uma variável $y$:

```python
x **= y # x = x ** y
```

```python
x = 2
x**=3
print(x)
```
