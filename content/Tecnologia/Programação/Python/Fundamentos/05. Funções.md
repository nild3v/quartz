## Introdução

Uma função é um bloco de código que executa uma determinada tarefa e permite reutilizá-la diversas vezes

Uma função pode ter parâmetros de entrada e uma saída, mas não é obrigatório:

```python showCodeRunner
# Atente-se ao nível de identação

# 'def' é a palavra reservada para indicar uma definição de função
def nome_da_funcao():
	# O 'pass' significa 'passar' ou 'ignorar'
	# Simplesmente não executará nada, é útil quando não se têm a 
	# implementação pronta e deseja adicioná-la no futuro, mas quer
	# deixar a função definida
	pass

# Imprimir o tipo da função
print(type(nome_da_funcao))
```

## Exemplos

Através do editor integrado do site, execute cada código abaixo:

```python showCodeRunner
# 'a' e 'b' são parâmetros
def soma(a, b):
	# 'return' é a palavra reservada para indicar o retorno de um valor
	return a + b

print(soma(2, 2))
```

```python showCodeRunner
# 'a' e 'b' são parâmetros
def subtracao(a, b):
	# 'return' é a palavra reservada para indicar o retorno de um valor
	return a - b

print(subtracao(2, 2))
```
