## Introdução

O primeiro código que todo iniciante de programação faz é o famoso e lendário "Hello World!", que consiste em imprimir (visualizar) esse texto no terminal

Nesse site há um editor de código integrado que permite executar Python diretamente no seu navegador. Mas se tiver um ambiente configurado na sua máquina, sinta-se a vontade em escolher onde quiser executar o código abaixo:

```python showCodeRunner
print("Hello World!")
```

Vamos entender o código:

- A palavra `print` é um nome de uma função, igual o `f` de `f(x)` na matemática

- Os parênteses que envolvem a frase, é o local onde colocamos os parâmetros da função. Assim como na matemática: `f(1)`, `f(10)`, ...

- O `"Hello World!"` é o parâmetro, no caso é um texto que na programação é chamado de `string`, que é obrigatoriamente delimitada por aspas

Veja um paralelo com a função matemática:

- `print(x)` -> `f(x)`
- `print("Hello World!")` -> `f(10)`
