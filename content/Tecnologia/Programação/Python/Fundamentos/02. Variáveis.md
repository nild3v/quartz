## Introdução

Uma variável é um rótulo ou um nome dado a um endereço de memória que referencia o dado ou valor armazenado

Para declarar uma variável em Python, primeiramente é necessário definir um nome que siga as seguintes regras:

- O nome da variável **não pode começar com número**

- O nome da variável **não pode ser igual a uma palavra reservada** (palavras já usadas pelo Python, como o `print`)

- O nome da variável **só pode conter: letras, números e sublinhados**

Após isso, o nome é seguido do sinal de igual (=) para indicar um atribuição, em seguida, passa-se o valor desejado:

```python
# ✅ Correto:
nome_da_minha_variavel = 1

# ✅ Correto:
outroNomeDeVariavel = 2

# ✅ Correto:
_variavel_1_ = 3

# ✅ Correto:
__varivel_2__ = 4

# ❌ Evite:
1variavel = 0

# ❌ Evite:
print = 1

# ❌ Evite:
class = 2
```

## Constantes em Python

Em Python diferentemente de outras linguagens de programação, não possui uma forma de declarar constantes, mas há uma nomenclatura usada para indicar que uma variável não deve ser modificada

O padrão usado declara o nome da constante totalmente em letra maiúscula:

```python
MINHA_CONSTANTE = 1
```

Lembrando que é apenas um nomenclatura, não significa que é uma constante de verdade, mas ao definir desse jeito, é esperado pelos desenvolvedores que não seja modificado
