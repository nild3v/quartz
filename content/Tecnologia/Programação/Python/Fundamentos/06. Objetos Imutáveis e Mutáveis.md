## Introdução

Em linguagens de programação há objetos imutáveis e mutáveis, ou seja, objetos que não se alteram e que se alteram após serem criados

Esse conceito é importante para entender um dos aspectos presentes não apenas no Python, mas em outras linguagens também

## Variáveis e Objetos

As variáveis não estão associadas a um valor ou tipo, elas são rótulos (nomes) dados a um ponto de referência na memória que contêm o objeto, seja mutável ou não

Caso você não compreenda isso, pode acabar criando um código que não funcione como o esperado

### Passagem por Valor

Veja o código abaixo:

```python
a = 1
b = a
```

O que você acha que acontece com a variável `a` ao modificarmos a variável `b`?

```python showCodeRunner
a = 1
b = a

# Acrescentando 1 ao valor de 'b'
b += 1

# Imprimindo as variáveis
print(a)
print(b)
```

Ao executar o código, perceba que ao alterar o valor de `b` não casou nenhum impacto na variável `a`, porque houve uma cópia desse valor

A variável `a` possui uma referência para um dado imutável (inteiro), logo, ao atribuir à variável `b`, as alterações feitas nela não refletem na variável `a`, como se tivesse feito uma cópia do valor

Isso ocorre em todos os objetos imutáveis

### Passagem por Referência

Ao contrário da passagem por valor, a passagem por referência pode ser problemática se não souber o que está fazendo

Vejamos o exemplo abaixo:

```python
a = [1, 2]
b = a
```

O que você acha que acontece ao adicionar um elemento em `b`?

```python showCodeRunner
a = [1, 2]
b = a

# Adicionando um elemento a lista
b.append(3)

# Imprimindo as variáveis
print(a)
print(b)
```

Ao atribuir `b` o valor de `a`, foi passado a referência na memória que `a` possuía, porque a lista é um dado mutável

Então, qualquer alteração em `b` refletirá em `a` e vice-versa

Caso queira evitar esse tipo de comportamento, terá que criar um cópia da lista:

```python showCodeRunner
a = [1, 2]
b = a.copy()

# Não reflete em a
b.append(3)

print(a)
print(b)
```

## Objetos Imutáveis

Aqui uma lista de objetos imutáveis em Python:

- Inteiros (int)
- Decimais (float)
- Booleanos (bool)
- Tuplas (tuple)
- Strings (str)
- Conjuntos Imutáveis (frozenset)
- Bytes (bytes)

## Objetos Mutáveis

Aqui uma lista de objetos mutáveis em Python:

- Listas (list)
- Dicionários (dict)
- Conjuntos (set)
- Bytearray (bytearray)
