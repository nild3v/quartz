Em Python as listas são uma coleção heterogênea (múltiplos tipos) de dados ordenados

Veja:

```python showCodeRunner
lista = [1, 2, 3, 4]

print(type(lista))
```

> [!warning] Observação
>
> Em Python, listas são diferentes de arrays. Diferentemente das listas, os arrays são homogêneos (único tipo)

## Indexação

É possível pegar um valor de uma lista a partir do seu índice de posição. O índice começa sempre em $0$ e indica a primeira posição:

```python showCodeRunner
lista = [4, 2, 1]

print(lista[0]) # Primeiro valor
print(lista[1]) # Segundo valor
print(lista[2]) # Terceiro valor
```

## Comprimento de uma Lista

No Python há uma função chamada `len` que permite saber o comprimento de uma lista:

```python showCodeRunner
lista = [1, 2, 3, 4]
comprimento = len(lista)

print(comprimento)
```

## Métodos

### .append()

Adiciona um valor à lista:

```python showCodeRunner
lista = [1, 2, 3, 4]

lista.append(5)
print(lista)
```

### .insert()

Insere um valor na lista em uma posição/índice específico:

```python showCodeRunner
lista = [1, 2, 3, 4]

lista.insert(1, 5) # Inserindo o número 5 no índice 1
print(lista)
```

### .remove()

Remove um valor da lista:

```python showCodeRunner
lista = [1, 2, 3, 4]

lista.remove(4)

print(lista)
```

Se a lista houver valores duplicados, o método irá remover o primeiro valor encontrado:

```python showCodeRunner
lista = [4, 4, 4, 4]

lista.remove(4)

print(lista)
```

### .clear()

Limpa/Remove todos os valores da lista:

```python showCodeRunner
lista = [1, 2, 3, 4]

lista.clear()
print(lista)
```

### .count()

Conta a quantidade de um determinado elemento na lista:

```python showCodeRunner
lista = [1, 2, 3, 4, 2, 2]

print(lista.count(2))
```

### .pop()

Remove o último valor da lista e o retorna:

```python showCodeRunner
lista = ['a', 'b', 'c', 'd']

# Removendo o último elemento e o armazenando em uma variável 
valor = lista.pop()

print(valor)
print(lista)
```

Mas também é possível remover um valor específico com base no índice:

```python showCodeRunner
lista = ['a', 'b', 'c', 'd']

# Removendo o primeiro elemento e o armazenando em uma variável 
valor = lista.pop(0)

print(valor)
print(lista)
```

### .copy()

Copia uma lista:

```python showCodeRunner
lista = ['a', 'b', 'c', 'd']

lista_copiada = lista.copy()

print(lista)
print(lista_copiada)
```

### .reverse()

Inverte a ordem de uma lista:

```python showCodeRunner
lista = ['a', 'b', 'c', 'd']

lista.reverse()

print(lista)
```
