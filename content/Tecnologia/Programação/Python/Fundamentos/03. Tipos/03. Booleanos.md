## Introdução

Os booleanos representam dois estados: **verdadeiro** ou **falso**. Além disso, estão relacionados com os números binários, 0 (falso) ou 1 (true)

O termo "boolean" origina-se da Álgebra Booleana, introduzida por George Boole

Veja:

```python showCodeRunner
valor_verdadeiro = True
valor_falso = False

print(type(valor_verdadeiro))
print(type(valor_falso))
```
