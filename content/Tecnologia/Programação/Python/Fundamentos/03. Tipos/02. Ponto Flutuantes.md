## Introdução

Os números ponto flutuantes na programação são os números reais que conhecemos na matemática

A expressão 'ponto flutuante' se refere ao modo que o ponto decimal (a vírgula) varia de posição de acordo com o número representado. Além disso é uma forma de representar números reais em computadores para que possam manipulá-los

Veja:

```python showCodeRunner
ponto_flutuante = 1.5

print(type(ponto_flutuante))
```

## Propriedades e Métodos

### .real

Retorna a parte real do número:

```python showCodeRunner
numero = 2.5

print(numero.real)
```

### .imag

Retorna parte imaginária do número:

```python showCodeRunner
numero = 2.5

print(numero.imag)
```

### .conjugate()

Retorna o conjugado do número:

```python showCodeRunner
numero = 2.5

print(numero.conjugate())
```

### .is_integer()

Verifica se o número é um inteiro:

```python showCodeRunner
numero = 2.5

print(numero.is_integer())
```

### .hex()

Converte o número em hexadecimal:

```python showCodeRunner
numero = 2.5

print(numero.hex())
```
