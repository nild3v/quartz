## Introdução

Uma string é uma sequência ou uma série de caracteres

O termo 'string' vem do inglês e significa 'corda' ou 'linha' em português, mas também pode indicar: **uma série de coisas organizadas em ou como se estivessem em uma linha**

Um texto é basicamente uma sequência/série de caracteres que estão organizados em uma ou múltiplas linhas. Assim como em cadernos usados na escola que possui linhas (strings)

Veja:

```python showCodeRunner
# Strings são delimitadas por aspas simples ou aspas duplas
string_1 = "Olá mundo!"
string_2 = 'Olá mundo!'

print(type(string_1))
print(type(string_2))
```

## Indexação

É possível pegar um caractere de uma sequência de caracteres a partir do seu índice de posição. O índice começa sempre em $0$ e indica a primeira posição:

```python showCodeRunner
texto = "abcdefga"

print(texto[0]) # Primeiro caractere
print(texto[1]) # Segundo caractere
print(texto[2]) # Terceiro caractere
print(texto[3]) # Quarto caractere
```

## Métodos

### .lower()

Retorna o texto completamente em minúsculo:

```python showCodeRunner
texto = "Olá mundo!"

print(texto.lower())
```

### .upper()

Retorna o texto completamente em maiúsculo:

```python showCodeRunner
texto = "Olá mundo!"

print(texto.upper())
```

### .islower()

Verifica se o texto todo está em minúsculo:

```python showCodeRunner
texto_1 = "olá mundo!"
texto_2 = "OLÁ MUNDO!"

print(texto_1.islower())
print(texto_2.islower())
```

### .isupper()

Verifica se o texto todo está em maiúsculo:

```python showCodeRunner
texto_1 = "olá mundo!"
texto_2 = "OLÁ MUNDO!"

print(texto_1.isupper())
print(texto_2.isupper())
```

### .capitalize()

Transforma a primeira letra contido no início da frase em maiúsculo:

```python showCodeRunner
texto = "como aprender a programar em python"

print(texto.capitalize())
```

### .replace()

Substitui uma sequência de caracteres por outra:

```python showCodeRunner
texto = "como aprender a programar em python"

print(texto.replace('como aprender', 'aprendendo'))
```

### .index()

Retorna o índice/posição do primeiro caractere de uma sequência de caracteres que deseja encontrar em um texto:

```python showCodeRunner
texto = "abcdefga"

print(texto.index('a')) # A letra 'a' está no índice 0
print(texto.index('b')) # A letra 'b' está no índice 1
print(texto.index('c')) # A letra 'c' está no índice 2
print(texto.index('abc')) # A sequência 'abc' começa no índice 0
```

### .isnumeric()

Verifica se uma string é uma sequência de caracteres numéricos:

```python showCodeRunner
textual = "abc"
numerico = "123"

print(textual.isnumeric())
print(numerico.isnumeric())
```
