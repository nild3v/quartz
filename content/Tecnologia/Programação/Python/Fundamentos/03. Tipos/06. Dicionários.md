## Introdução

Em Python os dicionários são uma coleção de pares chave-valor. Imagine um dicionário de verdade, onde há a **palavra** seguida do seu **significado**

A "palavra" seria a chave, o "significado" o valor:

```python showCodeRunner
dicionario = {
	'id': 1,
	'nome': 'João',
	'idade': 18,
	'curso': 'Engenharia de Software'
}

print(type(dicionario))
```

## Métodos

O dicionário possui diversos métodos ou funções que podem serem executadas

### .get()

Retorna o valor de acordo com a chave específicada:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

print(dicionario.get('nome'))
```

### .keys()

Retorna todas as chaves do dicionário:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

print(dicionario.keys())
```

### .values()

Retorna todos os valores do dicionário:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

print(dicionario.values())
```

### .items()

Retorna uma lista de pares chave-valor, onde cada par é um tupla contendo a chave e o valor respectivamente:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

print(dicionario.items())
```

### .clear()

Limpa todos os valores do dicionário:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

dicionario.clear()

print(dicionario)
```

### .copy()

Copia um dicionário:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

copia_do_dicionario = dicionario.copy()

print(dicionario)
print(copia_do_dicionario)
```

### .pop()

Remova e retorna o item do dicionário com base na chave especificada:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

print(dicionario.pop('nome'))
print(dicionario)
```

### .popitem()

Remove e retorna o último item do dicionário:

```python showCodeRunner
dicionario = {'nome': 'Maria'}

print(dicionario.popitem())
print(dicionario)
```
