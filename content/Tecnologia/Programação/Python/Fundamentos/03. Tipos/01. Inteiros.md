## Introdução

O inteiro em Python é um tipo numérico muito simples:

```python showCodeRunner
meu_numero_inteiro = 1

# Usamos novamente a função 'print', mas com um parâmetro de entrada diferente
# A função 'type' permite ver o tipo da variável

print(type(meu_numero_inteiro))
```

## Propriedades e Métodos

### .numerator

Retorna o numerador do número:

```python showCodeRunner
numero = 2

print(numero.numerator)
```

### .denominator

Retorna o denominador do número:

```python showCodeRunner
numero = 2

print(numero.denominator)
```

### .real

Retorna a parte real do número:

```python showCodeRunner
numero = 2

print(numero.real)
```

### .imag

Retorna a parte imaginária do número:

```python showCodeRunner
numero = 2

print(numero.imag)
```

### .conjugate()

Retorna o conjugado do número:

```python showCodeRunner
numero = 2

print(numero.conjugate())
```

### .is_integer()

Verifica se o número é um inteiro:

```python showCodeRunner
numero = 2

print(numero.is_integer())
```

### .to_bytes()

Converte o número para bytes:

```python showCodeRunner
numero = 2

print(numero.to_bytes())
```

### .bit_count()

Conta o número de bits "ligados":

```python showCodeRunner
numero = 2

print(numero.bit_count())
```

### .bit_length()

Retorna o comprimento mínimo necessário para representar o número em binário:

```python showCodeRunner
numero = 2

print(numero.bit_length())
```
