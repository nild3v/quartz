## Introdução

O [Google Collab](https://colab.research.google.com) é um serviço gratuito em nuvem baseado em Jupyter Notebooks que permite escrever e editar código em Python

A vantagem do Google Collab são os recursos generosos que ele fornece para executar algoritmos mais pesados gratuitamente, mas há um limite de uso

É recomendado para quem não quer ter que configurar um ambiente de desenvolvimento para programar em Python e não se preocupar em perder arquivos, porque fica salvo na nuvem. Entretanto é preciso ter uma conta Google para usar o serviço
