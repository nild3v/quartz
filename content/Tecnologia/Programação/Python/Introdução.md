Python é uma linguagem de programação muito popular, usada principalmente em Inteligência Artificial e Ciência de Dados

A simplicidade da linguagem serve como porta de entrada para muitos usuários iniciantes que desejam aprender a programar

Python é uma linguagem interpretada, ou seja, depende de um software para que possa ser executado. A linguagem base do Python é o C. Dessa forma, é possível adicionar extensões escritas em C ao Python

O nome da linguagem é originada de um programa da BBC chamado "Monty Python’s Flying Circus"
